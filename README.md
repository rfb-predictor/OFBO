# OFBO – Organic Flow Battery Ontology

The Organic Flow Battery Ontology (OFBO) is an ontology developed within the PREDICTOR project to provide a structured semantic representation of properties and concepts relevant to organic redox flow battery (RFB) electrolytes. It enables ontology-based data access (OBDA) to curated datasets from literature.  

## Repository Structure

- **database/csv** – Processed CSV files extracted from publications reporting molecular properties relevant to RFBs.  
- **database/h2** – H2 relational database containing the CSV datasets.  
- **ontology/OFBO.obda** – Ontop OBDA mappings for the ontology-based data access.  
- **ontology/OFBO.rdf** – The OFBO file.  
- **ontology/OFBO.properties** – Properties file for setting up Ontop with Protégé.
